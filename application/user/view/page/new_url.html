普通用户短链
用于普通的链接缩短访问、营销、推广等。
可以生成防红类的短链，和直接预览的短链。
可以使用第三方短链进行代理。
功能说明
支持链接到期功能：[适用于短时间需要访问的链接]
比如您的链接生成在1997年1月1日0时0分0秒，您可以设置链接的到期时间(支持到秒)
如果设定到1997年1月20日0时0分0秒到期
那么链接这个在1997年1月20日0时0分0秒将会暂停访问(并不会自动删除，用户可在后台删除，或者继续修改到期时间)
该功能可与其他功能一起使用，不会冲突
支持密码访问功能：[适用于个人分享，保密性的数据]
比如您设置的密码是：123456
那么当链接分享出去，只有知道这个密码的人才能继续访问
否则将会被拦截到原页面外
注意：此功能在浏览器内也同样适用！
该功能可与其他功能一起使用，不会冲突
支持第三方统计功能：[因为本站非专业数据统计站，所以允许接入第三方平台统计]
暂时只支持接入第三方支持js代码统计的平台，比如国内知名的：[cnzz]、[百度统计]
您可以添加统计代码，和预览链接，预览链接就是第三方平台的的链接
该功能可与其他功能一起使用，不会冲突
用户业务短链
用于大型企业级平台的链接缩短访问、营销、推广、活链、自定义、个性化、暂时性等。
可以生成防红类的短链，和直接预览的短链，还有限制访问数量型。
可以使用第三方短链进行代理访问。
功能说明
支持链接到期功能：[适用于短时间需要访问的链接]
比如您的链接生成在1997年1月1日0时0分0秒，您可以设置链接的到期时间(支持到秒)
如果设定到1997年1月20日0时0分0秒到期
那么链接这个在1997年1月20日0时0分0秒将会暂停访问(并不会自动删除，用户可在后台删除，或者继续修改到期时间)
该功能可与其他功能一起使用，不会冲突
支持密码访问功能：[适用于个人分享，保密性的数据]
比如您设置的密码是：123456
那么当链接分享出去，只有知道这个密码的人才能继续访问
否则将会被拦截到原页面外
注意：此功能在浏览器内也同样适用！
该功能可与其他功能一起使用，不会冲突
支持第三方统计功能：[因为本站非专业数据统计站，所以允许接入第三方平台统计]
暂时只支持接入第三方支持js代码统计的平台，比如国内知名的：[cnzz]、[百度统计]
您可以添加统计代码，和预览链接，预览链接就是第三方平台的的链接
该功能可与其他功能一起使用，不会冲突
支持链接访问次数限制功能：[适用于企业分享，活动、隐私类的链接]
因为考虑到业务短链的初衷设定是暂时性的，为了减少用户对短链的管理，特添加了此功能
使用说明：
短链属性选择[限制访问数量型短链],然后会出现让您输入访问数量的一个输入框，您只需要输入您希望的访问数量就行
访问数量计算是按照访问IP总计来计算的，当到达指定访问数量，将会暂停访问(并不会自动删除，用户可在后台删除，或者继续增加访问数量)
注意：此功能在浏览器内也同样适用！
该功能可与其他功能一起使用，不会冲突
支持活链功能：[自定义参数标识，自定义标识对应链接]
本功能需要慢慢看，因为会有点复杂
首先，生成业务短链页面会有个[参数自定义功能 ]的开关，我们先打开开关
然后会出现4给选项框
4个选项框分别为两个参数，看下图

1.代表这个功能的开关
2.基础的参数，如果不填，请关闭(1)的开关
3.第二个自定义参数，可以删除
4.可以继续添加一个参数
5.参数的唯一标识，标识必须唯一，因为涉及到判断，如果一样的话会导致链接跳转问题。且必须为1个数字或英文字符(可大小写)
6.参数标识对应的跳转链接，一个标识对应一个链接
7.可以删除添加的或已有的参数
举个例子：
比如您的主业务标识为:1，您的主链：https://www.baidu.com/
然后您的短链后缀是AEjCik
当您访问http://#.cn/1AEjCik
会跳转到您设置的主链接(就是第一个文本框)
那么您添加了一个自定义参数，标识为2，对应链接为https://www.sogou.com/
当您访问http://#.cn/1AEjCik2
会跳转到标识2对应的链接：https://www.sogou.com/
注意：此功能在浏览器内也同样适用！
该功能可与其他功能一起使用，不会冲突
支持短链向长链发送GET/POST功能：[可以直接在短链后面加入需要GET/POST的参数]
此方法支持多功能一起使用，并且支持多个提交参数 GET方法：
您的短链：http://#.cn/1AEjCik
您需要在后面加：?g=t&[您的参数]
例子：http://#.cn/1AEjCik?g=t&user=123&pass=123
POST方法：
您的短链：http://#.cn/1AEjCik
您需要在后面加：?p=t&[您的参数]
例子：http://#.cn/1AEjCik?p=t&user=123&pass=123
然后短链会以GET/POST的方式跳到您的主链上https://www.baidu.com/
注意：此功能在浏览器内也同样适用！
该功能可与其他功能一起使用，不会冲突
